# Juego de palabras

## 1. Abrir en VS Code

Descomprime la carpeta y en VS Code selecciona:

Archivo > Abrir carpeta > juego-palabras

## 2. Instalar dependencias

Abre la terminal de VS Code y ejecuta:

npm install

## 3. Configurar PostgreSQL

Crea una base llamada:

juego_palabras

Luego ejecuta el archivo database.sql.

## 4. Crear el archivo .env

Copia `.env.example`, renómbralo a `.env` y coloca tu contraseña de PostgreSQL.

## 5. Ejecutar

npm start

Después abre:

http://localhost:3000
